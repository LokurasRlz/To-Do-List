# To-Do-List

> This project is the structure for a To-Do-List application.


## 🧰 Built With

- HTML
- JS
- CSS
- Webpack Bundle

## Getting Started

### To get a local copy up and running follow these simple example steps:

- git clone https://github.com/LokurasRlz/To-Do-List.git
- cd To-Do-List
- npm install
- npm run start
